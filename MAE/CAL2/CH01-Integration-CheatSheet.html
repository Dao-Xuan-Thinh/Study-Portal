§1.1 Approximating Areas & Riemann Sums
Riemann Sum Setup
Definition
Partition [a, b] into n subintervals of width Δx = (b − a)/n. Choose sample points xi* in each subinterval [xi−1, xi].
Riemann Sum $$S = \sum_{i=1}^{n} f(x_i^*)\,\Delta x$$
  • Left endpoint: xi* = xi−1 → Ln
  • Right endpoint: xi* = xi → Rn
  • Midpoint: xi* = (xi−1+xi)/2 → Mn
Note
If f is increasing: Ln ≤ A ≤ Rn. If f is decreasing: Rn ≤ A ≤ Ln.
Definite Integral as a Limit
Definition
The definite integral of f over [a, b] is:
Definite Integral $$\int_a^b f(x)\,dx = \lim_{n\to\infty} \sum_{i=1}^{n} f(x_i^*)\,\Delta x$$
Theorem — Integrability
If f is continuous on [a, b] (or has finitely many jump discontinuities), then f is integrable on [a, b].
Useful Summation Formulas $$\sum_{i=1}^n i = \tfrac{n(n+1)}{2} \;\Big|\; \sum_{i=1}^n i^2 = \tfrac{n(n+1)(2n+1)}{6} \;\Big|\; \sum_{i=1}^n i^3 = \Bigl[\tfrac{n(n+1)}{2}\Bigr]^2$$

Average Value of a Function
Definition
The average value of f on [a, b]:
Average Value $$f_{\text{avg}} = \frac{1}{b-a}\int_a^b f(x)\,dx$$
Mean Value Theorem for Integrals
If f is continuous on [a, b], ∃ c ∈ (a, b) such that:
f(c) = (1/(b−a)) ∫ab f(x) dx
Example
Average value of f(x) = x² on [0, 3]: favg = (1/3)∫03 x² dx = (1/3)[x³/3]03 = 3
§1.3 The Fundamental Theorem of Calculus
FTC — Both Parts
FTC Part 1 — Derivative of Integral
If f is continuous on [a, b] and g(x) = ∫ax f(t) dt, then:

g′(x) = f(x) for all x ∈ (a, b)
Chain Rule Extension
d/dx [∫au(x) f(t) dt] = f(u(x)) · u′(x)
FTC Part 2 — Evaluation Theorem
If f is continuous on [a, b] and F is any antiderivative of f, then:

∫ab f(x) dx = F(b) − F(a) = [F(x)]ab
Key Insight
FTC bridges differentiation and integration — they are inverse operations.
Quick Example — FTC Part 2 $$\int_1^4 (2x+1)\,dx = \bigl[x^2+x\bigr]_1^4 = (16+4)-(1+1) = 18$$
§1.4 Integration Formulas & Net Change Theorem
Net Change Theorem
Theorem
The net change in a quantity F over [a, b] equals the integral of its rate of change:

F(b) − F(a) = ∫ab F′(x) dx
  • Displacement = ∫t₁t₂ v(t) dt
  • Distance = ∫t₁t₂ |v(t)| dt
  • Net change in population = ∫t₁t₂ r(t) dt

§1.5 The Substitution Rule (u-Substitution)
Substitution Rule
Theorem — u-Substitution
If u = g(x) is differentiable and f is continuous, then:

∫ f(g(x)) g′(x) dx = ∫ f(u) du
Method
  1. Choose u = g(x) (usually the "inner" function)
  2. Compute du = g′(x) dx
  3. Substitute and integrate in terms of u
  4. Back-substitute to express in terms of x
Example
∫ 2x·ex² dx: let u = x², du = 2x dx
= ∫ eu du = eu + C = ex² + C
Substitution for Definite Integrals
Theorem — Change of Limits
∫ab f(g(x)) g′(x) dx = ∫g(a)g(b) f(u) du
Strategy
When evaluating definite integrals, either:
(a) Change limits to u-values, or
(b) Back-substitute, then evaluate with original limits.
Example
∫02 x(x²+1)3 dx: u = x²+1, du = 2x dx
Limits: u(0)=1, u(2)=5
= (1/2)∫15 u3 du = (1/2)[u⁴/4]15 = (625−1)/8 = 78
Even/Odd Function Shortcuts
If f is even: ∫−aa f dx = 2∫0a f dx
If f is odd: ∫−aa f dx = 0
§1.6 Integrals of Exponential & Logarithmic Functions
Exponential Integrals
Natural Exponential $$\int e^x\,dx = e^x + C \qquad \int e^{kx}\,dx = \tfrac{1}{k}e^{kx} + C$$
General Base $$\int a^x\,dx = \frac{a^x}{\ln a} + C \quad (a>0,\; a\neq 1)$$
Logarithm Integral $$\int \frac{1}{x}\,dx = \ln|x| + C \qquad \int \ln x\,dx = x\ln x - x + C$$
Note on Absolute Value
∫ (1/x) dx = ln|x| + C handles both x > 0 and x < 0.

§1.7 Integrals Resulting in Inverse Trigonometric Functions
Standard Inverse Trig Integral Forms
Arcsine Form $$\int \frac{du}{\sqrt{a^2-u^2}} = \arcsin\!\left(\frac{u}{a}\right) + C$$
Arctangent Form $$\int \frac{du}{a^2+u^2} = \frac{1}{a}\arctan\!\left(\frac{u}{a}\right) + C$$
Arcsecant Form $$\int \frac{du}{u\sqrt{u^2-a^2}} = \frac{1}{a}\operatorname{arcsec}\!\left(\frac{|u|}{a}\right) + C$$
Usage Tip
Identify the constant a by matching the form; apply u-substitution first to reduce to a standard form.
Completing the Square
Technique
When integrand has a quadratic in denominator, complete the square to reveal a standard arctan/arcsin form.
Example
∫ 1/(x²+4x+8) dx
= ∫ 1/((x+2)²+4) dx
Let u = x+2, a = 2:
= (1/2) arctan((x+2)/2) + C
General Pattern — Arctan $$\int \frac{dx}{x^2+bx+c} \;\xrightarrow{\text{complete sq.}}\; \frac{1}{a}\arctan\!\left(\frac{u}{a}\right)+C$$
Watch out: Check if numerator is the derivative of the denominator — that yields ln|·| instead of arctan.
