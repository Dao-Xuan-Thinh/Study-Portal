§4.1 Basics of Differential Equations
Terminology & Classification
Differential Equation
An equation involving an unknown function y(x) and one or more of its derivatives.
TermMeaning
OrderHighest derivative present (y′ → 1st, y″ → 2nd, …)
DegreeExponent on the highest-order derivative term
Lineary and its derivatives appear to the first power, not multiplied together
NonlinearContains products of y·y′, or (y′)², or sin(y), etc.
AutonomousRight-hand side depends only on y, not on x: y′ = f(y)
Examples
y′ + 2y = 0 — 1st order, linear
y″ + y²= 0 — 2nd order, nonlinear
y′ = y(1−y) — 1st order, nonlinear, autonomous
General & Particular Solutions
General Solution
A family of solutions containing arbitrary constants (one per order). E.g., y = Ce−2x.
Particular Solution
Obtained by applying initial conditions to determine the constant(s).
Initial-Value Problem (IVP)
ODE + initial condition(s). Example:
y′ = −2y,   y(0) = 3
→ particular solution: y = 3e−2x
Existence & Uniqueness Theorem
If f and ∂f/∂y are continuous near (x₀, y₀), then the IVP y′ = f(x,y), y(x₀)=y₀ has a unique solution in some interval containing x₀.
§4.2 Direction Fields & Euler's Method
Direction Fields (Slope Fields)
Definition
A direction field for y′ = f(x,y): at each point (x,y), draw a short segment with slope f(x,y).
  • Nullclines: curves where f(x,y) = 0 (slopes are horizontal)
  • Equilibrium solutions: constant solutions where y′ = 0
  • Stable equilibrium: nearby solutions converge toward it
  • Unstable equilibrium: nearby solutions diverge from it
  • Semi-stable: solutions approach from one side, diverge from the other
Sketch Technique
For y′ = f(y) (autonomous), the direction field is independent of x — rows of segments have the same slope.
Euler's Method (Numerical)
Algorithm
Euler's method for y′ = f(x,y), y(x₀) = y₀ with step size h:
Euler's Method Formula $$x_{n+1} = x_n + h \qquad y_{n+1} = y_n + h\cdot f(x_n,\,y_n)$$
Example — y′ = x + y, y(0) = 1, h = 0.1
x₀=0, y₀=1: y₁=1+0.1·(0+1)=1.1
x₁=0.1, y₁=1.1: y₂=1.1+0.1·(0.1+1.1)=1.22
Accuracy
Euler's method has error O(h) per step. Smaller h → better approximation but more steps. The global truncation error is O(h) — first-order method.
§4.3 Separable Differential Equations
Separation of Variables
Form
A separable ODE has the form: dy/dx = g(x) · h(y)
5-Step Method
  1. Check for equilibrium solutions (h(y) = 0)
  2. Separate: dy/h(y) = g(x) dx
  3. Integrate both sides
  4. Solve for y (if possible)
  5. Apply initial condition to find constant C
Example
dy/dx = xy, y(0) = 2
dy/y = x dx → ln|y| = x²/2 + C
y = Aex²/2; y(0)=2 → A=2
y = 2ex²/2
Linear First-Order ODEs — Integrating Factor
Standard Form
Linear first-order ODE: dy/dx + P(x) y = Q(x)
Integrating Factor $$\mu(x) = e^{\int P(x)\,dx}$$
Method
  1. Write in standard form: y′ + P(x)y = Q(x)
  2. Compute μ(x) = e∫P dx
  3. Multiply both sides by μ: d/dx[μy] = μQ
  4. Integrate both sides: μy = ∫μQ dx + C
  5. Solve for y: y = (1/μ)[∫μQ dx + C]
Example
y′ + (2/x)y = x², P = 2/x, Q = x²
μ = e∫2/x dx = e2 ln x = x²
d/dx[x²y] = x⁴ → x²y = x⁵/5 + C → y = x³/5 + C/x²
§4.4 Applied Models
Exponential Models
ModelODESolution
Population growth dP/dt = kP P = P₀ekt  (k > 0)
Radioactive decay dN/dt = −λN N = N₀e−λt
Newton's cooling dT/dt = k(T−Tenv) T−Tenv = (T₀−Tenv)ekt
Compound interest dA/dt = rA A = A₀ert
Half-life & Doubling Time $$t_{1/2} = \frac{\ln 2}{\lambda} \qquad t_d = \frac{\ln 2}{k}$$
Carbon Dating
t = [ln(N/N₀)] / (−0.000121) where 0.000121 ≈ ln 2 / 5730 yr−1
Logistic Growth
Model
Logistic equation — population growth with carrying capacity K:
Logistic ODE $$\frac{dP}{dt} = rP\!\left(1 - \frac{P}{K}\right)$$
Logistic Solution $$P(t) = \frac{K}{1+Ae^{-rt}} \qquad \text{where }A = \frac{K-P_0}{P_0}$$
  • K = carrying capacity (horizontal asymptote)
  • r = intrinsic growth rate
  • P = K/2 is the inflection point (fastest growth)
  • P → K as t → ∞
Equilibrium Analysis
Equilibria at P = 0 (unstable) and P = K (stable).
