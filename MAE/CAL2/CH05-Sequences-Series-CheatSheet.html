§5.1 Sequences
Sequence Definitions & Notation
Definition
An infinite sequence {an}n=1∞ is a function from ℤ⁺ to ℝ.
  • Explicit: an = f(n) directly (e.g., an = 1/n)
  • Recursive: defined in terms of prior terms (e.g., an+1 = an/2)
  • Bounded above: ∃M such that an ≤ M for all n
  • Bounded below: ∃m such that an ≥ m for all n
  • Monotone increasing: an+1 ≥ an for all n
  • Monotone decreasing: an+1 ≤ an for all n
Limits of Sequences
Definition
limn→∞ an = L means an can be made arbitrarily close to L for all sufficiently large n. If the limit exists, the sequence converges; otherwise it diverges.
Squeeze Theorem
If an ≤ bn ≤ cn and lim an = lim cn = L, then lim bn = L.
Monotone Convergence Theorem
Every bounded monotone sequence converges. Specifically:
• Bounded above & increasing → converges to sup
• Bounded below & decreasing → converges to inf
Key Sequence Limits $$\lim_{n\to\infty} n^{1/n}=1 \;\Big|\; \lim_{n\to\infty}\!\left(1+\tfrac{1}{n}\right)^n = e \;\Big|\; \lim r^n=0\text{ if }|r|<1 \;\Big|\; \lim\frac{n^p}{e^n}=0$$
§5.2 Series: Definitions & Special Series
Series & Partial Sums
Definition
An infinite series: Σn=1∞ an = a₁ + a₂ + a₃ + …

The nth partial sum: Sn = Σk=1n ak

The series converges iff limn→∞ Sn = S (finite).
Divergence Test (nth Term Test)
If limn→∞ an ≠ 0, then Σan diverges.
⚠ Converse FALSE: lim an = 0 does not guarantee convergence (e.g., harmonic series).
Telescoping Series
Σ(bn − bn+1) = b₁ − limn→∞ bn+1
e.g., Σ 1/(n(n+1)) = Σ[1/n − 1/(n+1)] = 1
Geometric & p-Series
Geometric Series
Σn=0∞ arn = a/(1−r)  if |r| < 1
Diverges if |r| ≥ 1
Partial Sum of Geometric Series $$S_n = \frac{a(1-r^n)}{1-r} \quad (r\neq 1)$$
p-Series
Σn=1∞ 1/np:
• Converges if p > 1
• Diverges if p ≤ 1
Special Cases
p = 1: harmonic series Σ1/n — diverges
p = 2: Basel series Σ1/n² = π²/6 — converges
§5.3 The Integral Test & §5.4 Comparison Tests
Integral Test
Integral Test
If f is continuous, positive, and decreasing on [1,∞), and an = f(n), then:

Σn=1∞ an and ∫1∞ f(x) dx both converge or both diverge.
Remainder Estimate
If the series converges, the tail error after n terms satisfies:
∫n+1∞ f dx ≤ Rn ≤ ∫n∞ f dx
Example
Test Σ 1/(n ln²n): f(x) = 1/(x ln²x)
∫ dx/(x ln²x) = −1/ln x → converges (=1/ln 2)
Direct Comparison & Limit Comparison Tests
Direct Comparison Test
If 0 ≤ an ≤ bn for all n ≥ N:
• Σbn converges ⟹ Σan converges
• Σan diverges ⟹ Σbn diverges
Limit Comparison Test
If an, bn > 0 and limn→∞ (an/bn) = L > 0 (finite),
then Σan and Σbn both converge or both diverge.
Strategy
Compare to a geometric or p-series. Limit comparison is safer when direct comparison inequality is hard to establish.
Example — LCT
Σ (2n+1)/(n²+3): compare to Σ 1/n
lim (2n+1)/(n²+3) ÷ (1/n) = lim (2n²+n)/(n²+3) = 2 → same behavior → diverges
§5.5 Alternating Series & Absolute Convergence
Alternating Series Test (Leibniz)
Form
Alternating series: Σ (−1)n+1 an = a₁−a₂+a₃−…
Alternating Series Test (AST)
If (1) an > 0, (2) an+1 ≤ an (decreasing), and (3) lim an = 0, then the alternating series converges.
Alternating Series Estimation
|Error| = |S − Sn| ≤ an+1 (the first neglected term)
Alternating Harmonic Series
Σ (−1)n+1/n = 1−1/2+1/3−… = ln 2 ✓ converges
Absolute vs. Conditional Convergence
Absolute Convergence
Σan is absolutely convergent if Σ|an| converges.
Conditional Convergence
Σan is conditionally convergent if Σan converges but Σ|an| diverges.
Key Implication
Absolute convergence ⟹ convergence (but not vice versa).
Rearrangement
Riemann Rearrangement Theorem: A conditionally convergent series can be rearranged to converge to any real number!
§5.6 Ratio & Root Tests
Ratio Test
Ratio Test
Let L = limn→∞ |an+1/an|. Then:

• L < 1 → Σan converges absolutely
• L > 1 (or L = ∞) → Σan diverges
• L = 1 → test is inconclusive
Best Used For
Series involving factorials (n!), exponentials (an), or products of these.
Example
Σ n!/nn: L = lim (n+1)!/(n+1)n+1 · nn/n! = lim nn/(n+1)n = 1/e < 1 ✓ converges
Root Test & Test Summary
Root Test
Let L = limn→∞ |an|1/n. Then:

• L < 1 → Σan converges absolutely
• L > 1 (or L = ∞) → Σan diverges
• L = 1 → test is inconclusive
Best Used For
Series where an = f(n)n (nth power form).
